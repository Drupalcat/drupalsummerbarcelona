Drupal Summer Barcelona

Angular & drupal amigos con derecho a API

by Miguel Ángel and Álvaro J. Hurtado

Angular es uno de los frameworks de desarrollo frontend más populares que existen. Es muy sencillo desarrollar el frontend de una aplicación con angular y conectarla con diferentes backend.

La naturaleza de drupal 8, lo hace un entorno ideal para desarrollar el backend de una aplicación que se consuma mediante una API.

En esta charla enseñaremos la estructura de una pequeña aplicación de ejemplo en la que el frontend está hecho completamente en angular y la administración de contenidos y usuarios se gestiona mediante un drupal 8 con servicios web.

Drupal level:Basic, Language: Español
