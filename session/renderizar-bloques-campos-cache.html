Drupal Summer Barcelona

Como renderizar tus bloques y campos con buenas estrategias de cache

Featured speaker

by Jose Jiménez

En esta sesión me gustaía compartir mi experiencia en 3 conceptos:

  • El uso de Drupal Console para generar código y aprender del código generado: veremos algunos ejemplos del del uso de esta herramienta para generar algunos plugins.
  • Sistema de Plugins: No tendremos mucho tiempo para adentrarnos mucho en el sistema de plugins, pero si podemos comentar que diferencias hay respecto a los antiguos hooks y el actual sistema de Plugins en Drupal 8, veremos como podemos usar un bloque y algún field_formatter.
  • Sistema de Caché en Drupal 8: Cuando trabajamos con código custom en Drupal 8 tendremos un primer problema al estar todo cacheado, esta caché se puede desactivar para desarrollar pero posteriormente podremos tener otros tipos de problema si no pensamos en como invalidar esta caché para los render arrays que generemos, veremos como podemos solucionar estos problemas.

    • Drupal level: Beginner, Language: Español
